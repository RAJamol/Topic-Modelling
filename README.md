## News Topic Modeling on Indian Education
An end-to-end Natural Language Processing (NLP) pipeline that automatically collects news articles related to education in India, processes the text, and applies topic modeling (LDA) to uncover hidden themes and trends.

# Project Overview

This project demonstrates how to build a real-world NLP workflow by:

- Scraping news articles from Google News (RSS feed)
- Extracting full article content
- Cleaning and preprocessing text data
- Transforming text into numerical features
- Applying Latent Dirichlet Allocation (LDA) for topic modeling
- Interpreting key topics in education-related news

## Tech Stack

- **Python**
- **Selenium** – handling redirects & dynamic pages  
- **BeautifulSoup** – parsing RSS feeds  
- **Newspaper3k** – article extraction  
- **NLTK** – text preprocessing  
- **Scikit-learn** – vectorization & LDA  
- **Pandas / NumPy** – data handling

  ##  NLP Techniques Used

- Tokenization & normalization  
- Stopword removal (custom + default)  
- Lemmatization (WordNet)  
- Bag of Words (CountVectorizer)  
- Topic Modeling (LDA)
